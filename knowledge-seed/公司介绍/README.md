# 公司介绍材料 (`知识库/公司介绍/`)

公司基础介绍材料。供 `ps:rfp-analyze` 做战略分析时参考公司自身定位，以及 `ps:bid-draft` 生成"项目概述与响应总述"章节时引用。

## 放什么

- 公司简介 PDF / PPT / Markdown
- 品牌故事 / 企业文化文档
- 官方宣传册、画册
- 企业介绍视频截图 + 文字说明

## 命名约定

自由，建议按语言和用途分：

- `company-intro-zh.pdf` — 中文版公司简介
- `company-intro-en.pdf` — 英文版公司简介
- `brand-story.md` — 品牌故事
- `org-chart.png` — 组织架构图

## 格式

优先 Markdown（LLM 最易解析）；PDF / DOCX / PPTX 也支持。图片应配有文字说明。

## 谁引用它

- `ps:rfp-analyze`：写"评分杠杆分析"时需要知道公司自我定位
- `ps:bid-draft`：生成"项目概述与响应总述"、"公司简介"章节时直接引用内容
